Smart Basketball Hoop Scoring Tracker

April - May 2025

Cornell University

Developed an embedded system that tracks basketball shot accuracy in real-time using a KL46Z microcontroller. Leveraged a low-power infrared break-beam sensor (laser + photodiode) mounted under the hoop to detect made shots by measuring analog signal drops via ADC.

C Programming KL46Z Microcontroller ADC PWM LCD Interrupts Sensors Soldering
  • Architected interrupt-driven finite-state machine with four game states (IDLE, PREP, RUN, END)
  • Implemented real-time feedback: green LED flash, "Cha-Ching!" buzzer sound via PWM, and score updates on four-digit LCD
  • Built robust logic with countdown timers, false positive prevention, and seamless gameplay transitions
  • Successfully deployed and tested on physical hoop with minimal interference

TinyRV1 Processor & Accelerator Design

November - December 2024

Cornell University

Implemented a 32-bit single-cycle processor implementing the TinyRV1 RISC-V instruction set architecture using SystemVerilog. Developed the processor datapath and control unit with structural RTL design principles, incorporating multiplexers, adders, comparators, register files, and immediate generators.

SystemVerilog RISC-V FPGA RTL Design Digital Logic Intel Quartus Prime
  • Prototyped processor on Intel Cyclone V FPGA to run TinyRV1 assembly programs
  • Designed custom accumulate accelerator that outperformed general-purpose processor in cycle count and area efficiency
  • Conducted detailed hardware analysis including critical path identification and timing closure
  • Verified functional correctness through directed tests and trace-based simulations
